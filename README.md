## Project Overview
This project analyzes whether offensive performance (points, yards, turnovers) impacts team success in the NFL.

## Tools Used
- Python (Pandas)
- PostgreSQL
- Tableau
- Fast API

## Key Insight
Teams that score more points tend to win more games, while turnovers negatively impact success.
